# Martian Habitat Dashboard

## Overview
A browser-based simulation for managing life-support systems and crew health on Mars. Features real-time monitoring of oxygen, temperature, food, power, sleep, and wellness using simulated data.

## Key Features
- **Life Support Dashboard**: Real-time charts and alerts for environmental metrics.
- **Crew Management**: Profiles for 6 crew members with role-based access.
- **Terminal Interface**: Command-line interaction with privilege-based permissions.
- **ARIA AI Assistant**: Chat-based AI for diagnostics and support using Gemini API.
- **Responsive Design**: Dark, OLED-optimized UI with glass-morphism aesthetics.

## Technologies
- HTML, CSS, JavaScript
- Chart.js for visualizations
- Gemini 2.0 Flash API for AI
